login()

Logs in a user by validating the provided username and password. Returns userId and JWT token if the user is valid.
Parameters:
Name Type Description
req.body Object The request body JSON object.
Properties
Name Type Description
username string The username of the user.
password string The password of the user.
Throws:
  • If the password does not match.
    Type
    403
  • If the username or password is incorrect.
    Type
    404
  • If there is an error accessing the database.
    Type
    500
Returns:
JSON object with userId and token if login is successful.
Type
Object

register()

Registers a new user by creating an account with provided details. Returns userId and JWT token upon successful registration.
Parameters:
Name Type Description
req.body Object The request body JSON object.
Properties
Name Type Description
email string Required - The users email address.
name string Required - The users full name.
password string Required - The password of the user.
username string Required - The username of the user.
Throws:
  • An error message in JSON format if the email don't matches the configurable regex.
    Type
    400
  • An error message in JSON format if the email or username is already taken.
    Type
    403
  • An error message in JSON format if there is an error accessing the database.
    Type
    500
Returns:
JSON object with userId and token if registration is successful.
Type
Object
