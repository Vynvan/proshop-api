Module: Products

Methods

<inner> products/:id()

Retrieves the details of a specific active product by its ID. With the optional query param update=true, only the description is retrieved to update existing data on the client.
Parameters:
Name Type Description
req.body Object The request body JSON object.
Properties
Name Type Description
id number The ID of the product to retrieve.
req.query string The request querystring.
Properties
Name Type Argument Default Description
update boolean <optional>
 false Indicates whether to return the full product details or just the description.
Throws:
If there is an error accessing the database.
Type
500
Returns:
200 - JSON object containing the product. If req.query.update is true, only the id and full description is sent.
Type
Object

<inner> products/GET()

Retrieves a paginated list of active products from the database. Products only come with the first 50 chars of the description. Use "products/:id&update=true" to get the full description of the one product with the given id.
Parameters:
Name Type Description
req.query string The request querystring.
Properties
Name Type Argument Default Description
page number <optional>
 1 The page number for pagination.
limit number <optional>
 20 The number of products to return per page.
Throws:
An error message in JSON format if there is an error accessing the database.
Type
500
Returns:
200 - JSON object containing an array of products and pagination details.
Type
Object
